# Homebrew Tap

`brew tap drn/homebrew-tap`

## Available Formulae

* [Nerd LS](https://github.com/drn/nerd-ls)
  * `brew install drn/homebrew-tap/nerd-ls`

## Maintenance

*Updating Formula*:

```bash
brew edit <formula>
# update version
brew fetch <formula>
# update version & sha256 in this repository
```
